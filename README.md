# 📡 Redes_Cisco

Repositorio de prácticas de redes utilizando Cisco Packet Tracer, enfocado en el aprendizaje de fundamentos de redes, configuración de dispositivos y simulación de topologías.

# 📌 Descripción

Este repositorio contiene diversas prácticas de redes realizadas como parte del curso de redes.
Se implementan configuraciones básicas y avanzadas en entornos simulados para reforzar conceptos teóricos.

# 🎯 Objetivos
Comprender el funcionamiento de redes LAN y WAN
Aprender a configurar routers y switches
Practicar subnetting y direccionamiento IP
Implementar VLANs y routing básico
Desarrollar habilidades en simulación de redes

# 🧪 Contenido del repositorio
📁 Topologías de red (.pkt)
📁 Prácticas de VLANs
📁 Configuración de routing estático y dinámico
📁 Subnetting y direccionamiento IP
📁 Capturas de configuraciones

# 🖥️ Herramientas utilizadas
    - Cisco Packet Tracer
    - Conceptos de redes TCP/IP
    - Configuración de dispositivos Cisco (routers y switches)

# 📂 Estructura del proyecto
    - Trabajando en Ello

# 🚀 Aprendizajes
Configuración de redes desde cero
Uso de VLANs para segmentación de red
Enrutamiento entre redes
Diseño de topologías eficientes
Uso de simuladores de red profesional
